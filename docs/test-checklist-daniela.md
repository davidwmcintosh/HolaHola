# Daniela's Whiteboard Tools - Test Checklist

Use this checklist to verify each of Daniela's whiteboard tools is working correctly.

---

## Quick Reference

| Tool | Purpose |
|------|---------|
| WRITE | Display text on whiteboard |
| COMPARE | Show corrections with arrow |
| CLEAR | Clear all items |
| WORD_MAP | Word with synonyms/antonyms (async enriched) |
| IMAGE | Generate contextual image |
| GRAMMAR_TABLE | Conjugation/grammar tables |
| CONTEXT | Word usage in sentence |
| PHONETIC | Pronunciation breakdown |
| DRILL | Interactive exercises |
| PLAY | Replay audio |
| READING | Pronunciation guides (Asian languages) |
| STROKE | Animated stroke order |
| SCENARIO | Role-play setup |
| CULTURE | Cultural insights |
| SUMMARY | Lesson recap |
