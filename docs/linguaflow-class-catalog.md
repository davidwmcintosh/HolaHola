## Course Features

All LinguaFlow courses include:

- **AI Voice Tutoring:** Real-time conversation practice with Cartesia Sonic-3 TTS
- **ACTFL Alignment:** Progress tracking against World-Readiness Standards
- **Personalized Learning:** Adaptive difficulty based on student performance
- **Vocabulary Extraction:** Automatic word collection from conversations
- **Spaced Repetition:** Smart flashcard review system
- **Progress Tracking:** Visual ACTFL fluency charts
- **Cultural Tips:** Contextual cultural insights
