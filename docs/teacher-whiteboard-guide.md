# Teacher Guide: Using Whiteboard Tools

A guide for teachers on leveraging LinguaFlow's whiteboard system to enhance classroom instruction.

---

## Overview

The AI tutor uses a "whiteboard" system to display visual teaching aids during voice lessons. As a teacher, you should understand these tools to:

- Set appropriate expectations for your students
- Leverage whiteboard content in your syllabus design
- Understand what students see during AI tutoring sessions
- Provide supplementary instruction as needed

---

## How the Whiteboard Works

During voice lessons, the AI tutor can display visual elements on the student's screen:

1. **Automatic Display:** The tutor decides when to show visuals
2. **Strategic Use:** Tools appear at teaching moments, not constantly
3. **Interactive Elements:** Some tools require student interaction
4. **Lesson Flow:** Whiteboard enhances, doesn't replace, conversation

---

## Tool Categories

### Display Tools (Passive)

Students see information but don't interact:

| Tool | Best For | When to Expect |
|------|----------|----------------|
| WRITE | Vocabulary, key phrases | Introducing new words |
| PHONETIC | Pronunciation guides | Difficult words |
| COMPARE | Error correction | After student mistakes |
| IMAGE | Visual vocabulary | Concrete nouns |
| GRAMMAR_TABLE | Verb conjugations | Grammar instruction |
| CONTEXT | Usage examples | Showing word patterns |

### Interactive Tools (Active)

Students engage and respond:

| Tool | Best For | When to Expect |
|------|----------|----------------|
| DRILL (repeat) | Pronunciation practice | Speaking practice |
| DRILL (translate) | Active recall | Vocabulary review |
| DRILL (fill_blank) | Grammar practice | Form exercises |
| DRILL (match) | Vocabulary pairs | Quick review |
| PLAY | Audio replay | Pronunciation help |

### Asian Language Tools

Specialized for CJK languages:

| Tool | Purpose |
|------|---------|
| READING | Furigana/pinyin/romanization |
| STROKE | Animated character writing |

### Context Tools

Set up learning experiences:

| Tool | Purpose |
|------|---------|
| CULTURE | Cultural background |
| SCENARIO | Role-play setup |
| WORD_MAP | Vocabulary relationships |
| SUMMARY | Lesson recap |

## Classroom Applications

### Before AI Tutoring Sessions

1. **Set Expectations:** Explain that students will see visual aids
2. **Preview Tools:** Show examples of what they'll encounter
3. **Encourage Interaction:** Tell them to engage with drills
4. **Cultural Context:** Prepare for cultural information they'll learn

### During Class Time

Use whiteboard knowledge to:

- **Reference Tools:** "Remember the conjugation table the tutor showed?"
- **Reinforce Learning:** "Let's practice those matching pairs again"
- **Extend Content:** "The tutor mentioned tipping culture; let's discuss more"
- **Address Struggles:** "If stroke order is hard, practice with paper"

### After AI Sessions

Follow up with:

- **Vocabulary Review:** Quiz on words shown with WRITE/IMAGE
- **Grammar Practice:** Worksheets matching GRAMMAR_TABLE content
- **Cultural Discussion:** Expand on CULTURE topics
- **Writing Practice:** Character practice from STROKE demonstrations

## Syllabus Design Tips

### Aligning with Whiteboard Tools

When designing your syllabus, consider:

| Lesson Focus | Useful Tools | Your Support |
|--------------|--------------|--------------|
| New Vocabulary | IMAGE, WRITE, WORD_MAP | Flashcard exercises |
| Grammar Point | GRAMMAR_TABLE, CONTEXT | Worksheet practice |
| Pronunciation | PHONETIC, PLAY, DRILL | Speaking activities |
| Cultural Topic | CULTURE, SCENARIO | Discussion questions |
| Character Writing | STROKE, READING | Handwriting practice |

### Sequencing for Tool Support

**Week 1-2: Foundation**
- Focus on WRITE, IMAGE for vocabulary
- Simple DRILL (match) for review
- PHONETIC for pronunciation foundation

**Week 3-4: Building**
- Add GRAMMAR_TABLE for structures
- CONTEXT for usage patterns
- DRILL (fill_blank) for practice

**Week 5+: Expanding**
- CULTURE for deeper understanding
- SCENARIO for role-play
- WORD_MAP for vocabulary networks

## Best Practices for Teaching

### Supplement, Don't Duplicate

The AI tutor handles:
- Real-time visual feedback
- Interactive pronunciation practice
- Immediate error correction

You should provide:
- Group activities
- Extended cultural discussion
- Personalized attention
- Writing/reading practice

### Monitor Student Progress

Watch for students who:
- Struggle with drills (may need prerequisite review)
- Skip interactive elements (encourage participation)
- Don't use Asian language tools (may need writing instruction)
- Miss cultural context (expand in class)

### Common Issues & Solutions

| Issue | Likely Cause | Solution |
|-------|--------------|----------|
| Student skips drills | Anxiety or frustration | Build confidence with easier content |
| Poor pronunciation | Not using PLAY button | Encourage audio replay |
| Character confusion | Skipping STROKE practice | Assign writing homework |
| Grammar errors persist | Not reviewing GRAMMAR_TABLE | Paper conjugation practice |

## Class Settings Configuration

As a teacher, you can configure how the AI tutor behaves in your classes:

### Tutor Freedom Levels

| Level | Description | Whiteboard Use |
|-------|-------------|----------------|
| Strict | Follows syllabus exactly | Controlled, syllabus-aligned |
| Guided | Stays close to syllabus | Moderate flexibility |
| Flexible | Can explore related topics | More varied tool use |
| Open | Student-led exploration | Full tool availability |

### Recommendations by Class Type

**Beginner Classes:**
- Use Strict or Guided mode
- Focus on WRITE, IMAGE, simple DRILL
- Limit cultural tangents initially

**Intermediate Classes:**
- Use Guided or Flexible mode
- Add GRAMMAR_TABLE, CONTEXT
- Include CULTURE for depth

**Advanced Classes:**
- Use Flexible or Open mode
- All tools available
- Emphasize WORD_MAP, SCENARIO

## Tracking Whiteboard Effectiveness

### What to Observe

In student progress:
- Vocabulary retention after IMAGE use
- Pronunciation improvement with PHONETIC/PLAY
- Grammar accuracy after GRAMMAR_TABLE
- Writing improvement after STROKE

### Questions to Ask Students

- "What visual aids helped you most?"
- "Did the matching drills help you remember vocabulary?"
- "Was the cultural information interesting?"
- "Did the stroke order animations help your writing?"
