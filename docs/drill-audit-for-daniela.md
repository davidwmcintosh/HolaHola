The drill library is heavily weighted toward **novice-level audio/speaking drills** (listen_repeat, number_dictation). Higher ACTFL levels and grammar/vocabulary reinforcement drills are significantly underrepresented.

## Part 1: Drill Types Overview

### Available Drill Types
| Type | Description | Use Case |
|------|-------------|----------|
| `listen_repeat` | Listen to audio, repeat pronunciation | Pronunciation, speaking fluency |
| `number_dictation` | Hear numbers, write them | Listening comprehension, numbers |
| `translate_speak` | See native language, speak target | Production, vocabulary recall |
| `fill_blank` | Grammar fill-in-the-blank | Grammar structures, conjugation |
| `matching` | Vocabulary matching exercises | Vocabulary retention |

### Key ACTFL Observations

1. **Novice_low has 99%+ of listen_repeat and number_dictation drills**
   - This makes sense pedagogically (building pronunciation habits early)
   - But intermediate/advanced students have almost no pronunciation drills

2. **Fill_blank dominates at higher levels**
   - Advanced levels: fill_blank is 90%+ of drills
   - This shifts focus to grammar at higher levels

3. **No number_dictation above novice_low**
   - Numbers are only practiced at the earliest stage

4. **translate_speak drops significantly above novice_low**
   - Only 100-200 drills across all intermediate/advanced combined

## Part 5: Identified Gaps

### Critical Gaps

| Gap | Impact | Recommendation |
|-----|--------|----------------|
| No pronunciation drills above novice_low | Advanced students lose speaking practice | Add listen_repeat drills for intermediate+ with complex sentences |
| Asian languages lack translate_speak | Students can't practice production | Create character-aware translation drills |
| Very few matching drills (155 total) | Weak vocabulary reinforcement | Expand matching drills across all levels |
| Advanced classes nearly empty (8-13 lessons) | Limited content for committed learners | Prioritize advanced content creation |

### Distribution Imbalance

**Current vs Suggested Distribution**

| Drill Type | Current | Suggested Target |
|------------|---------|------------------|
| listen_repeat | 57% | 35-40% |
| number_dictation | 31% | 10-15% |
| translate_speak | 12% | 20-25% |
| fill_blank | 1.3% | 15-20% |
| matching | 0.08% | 5-10% |

## Part 6: Questions for Daniela

1. **Asian Language Strategy**: Is the minimal translate_speak content for Japanese/Korean/Mandarin intentional, or should we create adapted drills that account for writing system differences?

2. **Intermediate/Advanced Pronunciation**: Should we create listen_repeat drills for higher levels focusing on nuanced pronunciation (intonation, emotion, formal/informal registers)?

3. **Number Dictation Ceiling**: Is it correct that number practice stops at novice_low? Or should we add advanced number drills (large numbers, decimals, phone formats, addresses)?

4. **Grammar Drill Balance**: The shift to mostly fill_blank at advanced levels is stark. Is this the right pedagogical approach, or do we need more variety?

5. **Matching Drill Expansion**: With only 155 matching drills total, should this be a priority for vocabulary reinforcement, or does conversational practice with you adequately cover vocabulary retention?
