## Getting Started

### Teacher Role

As a teacher, you have access to:
- **Class creation and management** - Create classes, enroll students
- **Syllabus customization** - Adapt curriculum templates for your classes
- **Progress monitoring** - Track individual and class-wide achievement
- **Assignment creation** - Assign practice tasks with due dates
- **ACTFL coverage analysis** - See how your syllabus aligns with standards

### Accessing Teacher Features

1. Navigate to **My Classes** in the sidebar
2. Create a new class or select an existing one
3. Use tabs to access Syllabus, Students, Assignments, and Analytics

## Class Management

### Creating a Class

1. Go to **My Classes** → **Create Class**
2. Fill in:
   - **Class name** (e.g., "Spanish 101 - Fall 2025")
   - **Language** being taught
   - **Target level** (ACTFL proficiency)
   - **Description** (optional)
3. Select a **curriculum template** to start with
4. Click **Create**

### Student Enrollment

**Join Codes:**
- Each class has a unique 6-character join code
- Share this code with students
- Students enter it at **Classes** → **Join Class**

**Managing Students:**
- View enrolled students in the **Students** tab
- See individual progress and activity
- Remove students if needed

### Tutor Freedom Levels

Control how strictly the AI tutor follows your syllabus:

| Level | Description | Best For |
|-------|-------------|----------|
| **Strict** | Follows syllabus exactly | Beginners, structured courses |
| **Guided** | Stays close to syllabus | Most classes |
| **Flexible** | Can explore related topics | Intermediate+ students |
| **Open** | Student-led exploration | Advanced, independent learners |

## Syllabus Builder

### Overview

The Syllabus Builder lets you customize curriculum for your specific class after selecting a template.

### Features

| Feature | Description |
|---------|-------------|
| **Drag-and-drop** | Reorder units and lessons |
| **Add lessons** | Create custom lessons with name, description, content |
| **Edit lessons** | Modify existing lesson content for local needs |
| **Remove lessons** | Soft-delete lessons (can be restored) |

### Workflow

1. **Start from template** - Choose a curriculum template when creating class
2. **Customize** - Use the Syllabus tab to modify for your needs
3. **Reorder** - Drag units and lessons to adjust pacing
4. **Add custom content** - Create lessons specific to your class
5. **Edit** - Adapt lesson descriptions and content

**Note:** Your changes are class-specific; templates remain unchanged for others.

### ACTFL Standards Coverage Panel

The Syllabus Builder includes real-time ACTFL analysis:

- **Overall coverage percentage** with progress bar
- **Category breakdown:** Interpersonal, Interpretive, Presentational
- **Level-by-level Can-Do tracking** (Novice through Advanced)
- **Visual indicators:** Covered (✓) vs uncovered (○)
- **Expandable UI** for detailed exploration

Use this to ensure your syllabus adequately covers required standards.

## Student Progress Monitoring

### Individual Progress

For each student, you can see:
- **Proficiency level** - Current ACTFL level
- **Vocabulary mastered** - Words learned and retention
- **Conversation time** - Total voice practice minutes
- **Syllabus progress** - Lessons completed vs total
- **Assignment completion** - Submitted and pending

### Class-Wide Analytics

- **Proficiency distribution** - How many students at each level
- **Engagement metrics** - Active students, average practice time
- **Syllabus completion** - Class-wide progress through curriculum
- **Early completions** - Students ahead of schedule

### Organic Progress Detection

HolaHola automatically recognizes when students cover curriculum topics during free conversation practice.

**How It Works:**
1. Student has conversations with AI tutor
2. System analyzes topics, vocabulary, grammar demonstrated
3. If 80%+ competency detected, student can complete lesson early
4. Teacher reviews and approves

**Verification Workflow:**
1. Go to **Class Management** → **Organic Progress** tab
2. Review students with demonstrated competency
3. See detailed breakdowns:
   - Topic coverage percentage
   - Vocabulary words mastered
   - Grammar concepts demonstrated
   - Pronunciation scores
4. Click **Verify Completion** to approve
5. Add optional notes about progress

**Benefits:**
- Students progress faster
- You save time on assessment
- Personalized learning paths
- Higher engagement

## Whiteboard Tools

### Overview

During voice lessons, the AI tutor displays visual aids on a "whiteboard." Understanding these tools helps you:
- Set appropriate expectations for students
- Leverage content in syllabus design
- Provide supplementary instruction

### Tool Categories

**Display Tools (Passive):**

| Tool | Best For | When It Appears |
|------|----------|-----------------|
| WRITE | Vocabulary, key phrases | Introducing new words |
| PHONETIC | Pronunciation guides | Difficult words |
| COMPARE | Error correction | After student mistakes |
| IMAGE | Visual vocabulary | Concrete nouns |
| GRAMMAR_TABLE | Verb conjugations | Grammar instruction |
| CONTEXT | Usage examples | Showing word patterns |

**Interactive Tools (Active):**

| Tool | Best For | When It Appears |
|------|----------|-----------------|
| DRILL (repeat) | Pronunciation | Speaking practice |
| DRILL (translate) | Active recall | Vocabulary review |
| DRILL (fill_blank) | Grammar practice | Form exercises |
| DRILL (match) | Vocabulary pairs | Quick review |
| PLAY | Audio replay | Pronunciation help |

**Asian Language Tools:**

| Tool | Purpose |
|------|---------|
| READING | Furigana/pinyin/romanization |
| STROKE | Animated character writing |

**Context Tools:**

| Tool | Purpose |
|------|---------|
| CULTURE | Cultural background |
| SCENARIO | Role-play setup |
| WORD_MAP | Vocabulary relationships |
| SUMMARY | Lesson recap |

### Classroom Applications

**Before AI Sessions:**
- Explain that students will see visual aids
- Preview what they'll encounter
- Encourage interaction with drills

**During Class:**
- Reference tools: "Remember the conjugation table the tutor showed?"
- Reinforce learning: "Let's practice those matching pairs"
- Extend content: "The tutor mentioned tipping culture; let's discuss"

**After Sessions:**
- Quiz on vocabulary from WRITE/IMAGE tools
- Worksheets matching GRAMMAR_TABLE content
- Cultural discussions expanding on CULTURE topics
- Character writing practice from STROKE demonstrations

### Aligning Syllabus with Tools

| Lesson Focus | Useful Tools | Your Support |
|--------------|--------------|--------------|
| New Vocabulary | IMAGE, WRITE, WORD_MAP | Flashcard exercises |
| Grammar Point | GRAMMAR_TABLE, CONTEXT | Worksheet practice |
| Pronunciation | PHONETIC, PLAY, DRILL | Speaking activities |
| Cultural Topic | CULTURE, SCENARIO | Discussion questions |
| Character Writing | STROKE, READING | Handwriting practice |

## Assignment System

### Creating Assignments

1. Go to class → **Assignments** tab
2. Click **Create Assignment**
3. Fill in:
   - **Title** and **Description**
   - **Type:** Practice, Homework, Quiz, or Project
   - **Due date**
   - **Points** (optional)
   - **Instructions**

### Assignment Types

| Type | Best For |
|------|----------|
| **Practice** | Low-stakes conversation practice |
| **Homework** | Regular learning activities |
| **Quiz** | Assessment of specific topics |
| **Project** | Longer-term, multi-skill tasks |

### Grading Workflow

1. View submitted assignments in **Assignments** tab
2. Click on a submission to review
3. Add score and feedback
4. Mark as graded

### Linking Drills to Conversations

The system can auto-link drill lessons to related conversations:
- "Greetings Drill" → "Greetings Conversation"
- "Numbers Drill" → "Numbers Conversation"

Students must complete the drill (70%+ mastery) before unlocking the conversation lesson.

## Best Practices

### Supplement, Don't Duplicate

**The AI tutor handles:**
- Real-time visual feedback
- Interactive pronunciation practice
- Immediate error correction
- 24/7 availability

**You should provide:**
- Group activities and discussions
- Extended cultural exploration
- Personalized attention
- Writing/reading practice
- Human connection and motivation

### Monitor for Issues

Watch for students who:
- **Struggle with drills** → May need prerequisite review
- **Skip interactive elements** → Encourage participation
- **Don't use Asian language tools** → May need writing instruction
- **Miss cultural context** → Expand in class

### Common Issues & Solutions

| Issue | Likely Cause | Solution |
|-------|--------------|----------|
| Student skips drills | Anxiety or frustration | Build confidence with easier content |
| Poor pronunciation | Not using audio replay | Encourage PLAY button use |
| Character confusion | Skipping stroke practice | Assign writing homework |
| Grammar errors persist | Not reviewing tables | Paper conjugation practice |

### Pacing Recommendations

**Week 1-2: Foundation**
- Focus on vocabulary display tools (WRITE, IMAGE)
- Simple matching drills for review
- Phonetic guides for pronunciation

**Week 3-4: Building**
- Add grammar tables for structures
- Context sentences for patterns
- Fill-in-the-blank drills

**Week 5+: Expanding**
- Cultural context for depth
- Role-play scenarios
- Word maps for vocabulary networks

### Questions to Ask Students

- "What visual aids helped you most?"
- "Did the matching drills help you remember vocabulary?"
- "Was the cultural information interesting?"
- "Did stroke order animations help your writing?"
