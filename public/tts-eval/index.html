What to listen for

Focus on the highlighted target words in each test. Does the provider pronounce them in the correct language? The critical test is "come stai" (Italian) - Cartesia has been pronouncing "come" as English and "stai" as "stay." Also listen for accent consistency - does the voice maintain a natural tutor personality across languages?

Methodology notes

Latency: Measured as full HTTP round-trip (not streaming TTFB). In production we'd use WebSocket streaming for both, so real latency would be lower. Cartesia timestamps show 0 because the bytes API doesn't return them (only WebSocket API does). ElevenLabs timestamps are derived from character-level alignment data. Voices are not matched 1:1 - Cartesia uses language-specific voices, ElevenLabs uses a single premade voice. This eval focuses on pronunciation correctness, not voice personality matching.
